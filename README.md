## Analyzing Community-Level Homogeneity and Heterogeneity Across Reddit Communities

<img src="https://i.imgur.com/KVnfNZs.jpeg" alt="tsne" width="250"/>

<img src="https://i.imgur.com/2cbJTG2.png" alt="Network Graph" width="250"/>

[Report Link](https://drive.google.com/file/d/1JoT3cRAyd5dhzUA9Ays_m9dNpCEFQE6P/view)

Social media communities reflect real-world sentiments, and analyzing their topical structure offers insight into information flow and diversity. Using BERTopic, we analyzed homogeneity and heterogeneity across 34 subreddits by collecting ~1,000 hot posts per community and extracting dominant themes. We represented each community with one-hot topic vectors and applied cosine similarity, clustering, and t-SNE/UMAP visualizations. Our results reveal clear differences in topic focus, uncovering latent community structures and patterns of homophily and heterophily.


## CommentCrafter - Full-Stack LLM-Powered Training Data Generation Platform for E-Commerce Sites

<img src="https://i.imgur.com/rY4EfVb.png" alt="CommentCrafter" width="250"/>

<img src="https://i.imgur.com/ANqYS4O.png" alt="CommentCrafter" width="250"/>

[Link to GitHub](https://github.com/IvanHornung/CommentCrafter?tab=readme-ov-file)
[Link to site](https://cc-web-client-737754814579.us-central1.run.app/)

CommentCrafter is a full-stack, LLM-powered platform that generates bulk synthetic user comments for training e-commerce comment filters. Users submit a product URL, desired comment count, and ‚Äúpollution‚Äù level; the system infers product context, produces mock UGC, and assigns relevancy/offensiveness scores. Results can be exported (CSV/JSON/XML) and saved to view generation history. Built with a Flask + Firestore backend, Next.js/TypeScript frontend, and deployed via Docker on Google Cloud Run using Vertex AI (Gemini).
